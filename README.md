# Travel Destination Picker & Booking App (Beginner Friendly)
## What's included
- backend/ (Node + Express)
- frontend/ (React)
- sql/ (database schema + sample data)

## Quick start
1. Import `sql/travel_app.sql` into your MySQL server.
2. Backend:
   - cd backend
   - npm install
   - node server.js
3. Frontend:
   - cd frontend
   - npm install
   - npm start

Backend runs on port 5000, frontend on 3000 by default.
